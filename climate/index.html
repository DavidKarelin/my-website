About This Project

This climate artifact was developed by David Karelin. It uses APIs from Open-meteo and the NOAA to get current climate data and compare it to historical climate data, based in 2025. Baseline values are either based on current averages for the data at the time of project completion or gathered from APIs. You can see how long this project has been in operation for at the top banner. All values are calculated for La Jolla, San Diego. Temperature data is collected with Open-meteo's archival weather API, atmospheric CO2 data is collected with an Open-meteo forcecast model (CAMS), and sea level data is collected with the NOAA CO-OPS API, collecting data from the La Jolla station. All baseline values are based in 2025, except for while in 2025, which uses 2024 as its baseline. You can hover over a bar to see its value.
